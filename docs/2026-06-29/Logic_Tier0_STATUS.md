### Verified (live)
- **Renderer / UI** — the reused DAW-agnostic Electron drop-UI drives the export via
  the Flask backend; only file-type labels were swapped. Cursor never moves; native
  sample rate; project never saved.
- **DialogGuard — C9 + fail-safe + C7 all live.**
  - **C9** (audio-interface-unavailable): detected → auto-clicks **OK** →
    cleared → ready. Re-observed repeatedly during the P8 focus-diagnosis runs
    (`rule=audio_interface_unavailable -> click button='OK'`).
  - **fail-safe**: any unrecognized/never-safe dialog → **PAUSE-notify** (never
    guesses a click); recognized terminal dialogs (e.g. *export failed*) → click the
    safe **OK**, then abort the job cleanly (never saved).
  - **C7** (read-only / permission-denied output): surfaces on the PAUSE/fail path.
  - Engine coverage: **49 DialogGuard unit tests green** (`tests/`), incl. the
    post-Export mid-wait scan that stops a late dialog from hanging the bounce-wait.
- **Notifications** — a **single native macOS notification** fires only on a critical
  failure (guard PAUSE / terminal dialog / render that can't complete), carrying
  `{project, reason}`. Healthy runs are **silent** (no `[[EXPORT_FAILURE]]` marker).
- **In-UI inbox** — the owner-approved net-new UI surfaces the same
  `{project, reason}` signal (the one deliberate exception to "reuse the UI 100%").
- **Two-set output** — exactly `01_With_FX` (plugins on) + `02_Raw` (Bypass Effect
  Plug-ins on → dry), each per-track; the T11 guard proves Raw really differs from Wet.
- **Single-zip deliverable (verify-then-delete)** — the job produces one
  `<project>.zip`; the un-zipped source folder is deleted **only after** the zip is
  verified (exists, non-zero, valid archive, every expected stem present). On any
  verification failure the folder is kept and the failure surfaces.
- **Focus restore follows the user's window** — the restore target is the user's most
  recent **real** working window (tracked continuously; excludes Logic **and** our own
  launcher app, which main.js passes to the backend). Focus returns there after the
  masked flicks and after quit, with a belt-and-braces restore off Logic at job end.

## Known residuals — deferred to Tier 1 (accepted for Tier 0)

Tier 1 (off-screen virtual display + keyboard event-tap) is built **once, shared**
across all three DAWs and is explicitly **last** (roadmap §2). Until then:

- **(i) Launch/load focus + screen grab (~26 s).** From `open -g` until the first
  masked flick, Logic self-activates on project load and is visible/frontmost for the
  load window (~26 s on the test fixture). Tier 0 backgrounds it as soon as the flow
  reaches the first flick; full invisibility during load needs the Tier 1 virtual
  display.
- **(ii) Sub-second keyboard leak during the masked flick.** The two irreducible
  chords (⌘⇧G / Return) require Logic frontmost for that instant; a keystroke aimed at
  Logic in that ~sub-second window would land there. Eliminated by the Tier 1 keyboard
  event-tap ("keyboard numbing"). No Tier 0 action (per roadmap §5).
- **(iii) Load-window visibility.** Logic's project/load window is visible on screen
  during load (same root cause as (i)); hidden only under the Tier 1 virtual display.

## How to run the certification (hands-off)

Logic **closed** first. From your **normal frontmost app** (e.g. Terminal / TextEdit /
Claude), leave the machine alone during the render:

```
# Terminal 1 — backend + UI (leave running; watch its stdout for C9 / [SCAN] / failures)
npm start

# Terminal 2 — trigger + assert the pipeline, then hands-off
python3 tools/tester.py
```

Static / preflight-only (no backend, no render — validates the checks themselves):

```
python3 tools/tester.py --preflight
```

`tools/tester.py` writes its results into the gitignored `STATUS.md` run-report and
prints ALL GREEN / failures; then confirm in Terminal 1: a
`[DialogGuard] … rule=audio_interface_unavailable -> click button='OK'` line (C9),
**no** `[[EXPORT_FAILURE]]` line, and the `[SCAN]` totals (last `[SCAN]` line =
cumulative across load + both passes).
